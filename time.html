Golang Time Converter

Golang provides a Time package that contains functionality for measuring and displaying time. One of the primary data structures that is consumed and returned by functions in the Time package is the Time type. This structure contains several private fields that are used internally and are normally invisible. However, sometimes when stepping through code using the debugger, it would be helpful to have the ability to decode the internal values and display the date/time in a human-readable format. This utility serves that purpose. Enter the wall and ext values below to convert to UTC.
